AWS Services Demonstrated

🛡 WAF

Web Application Firewall protecting against common exploits like SQL injection, XSS, and rate limiting malicious requests.

⚖ ALB

Application Load Balancer distributing traffic with health checks, SSL termination, and target group management.

💻 EC2

t4g.nano Graviton instance running FastAPI with Gunicorn, demonstrating cost-effective ARM-based computing.

🌐 Route 53

DNS management with alias records pointing to CloudFront distributions for both static and API endpoints.

⚡ CloudFront

Global CDN with edge caching, Origin Access Control (OAC) for S3, and custom origin for ALB.

🗃 S3

Static website hosting with bucket policies, encryption at rest, and CloudFront integration via OAC.

API Endpoints

  • GET /api/health Health check for ALB target group
  • GET /api/hello Returns "Hello World" as JSON
  • GET /api/time Current Philippine time as JSON
